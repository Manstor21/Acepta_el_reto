# Acepta el Reto

Resolución diaria de problemas de [¡Acepta el reto!](https://aceptaelreto.com), el juez de programación en línea de la Universidad de Valladolid.

## Estructura

Cada problema se organiza en su propia carpeta siguiendo el formato `ID.Nombre_del_problema` dentro del volumen que le corresponde:

```
001-100/
  100.Constante de Kaprekar/
    Kaprekar.java
  101.Cuadrados diabolicos y esotericos/
    CuadradosMagicos.java
101-200/
  ...
```

## Progreso

| Volumen | Rango | Resueltos |
|---------|-------|-----------|
| 1 | 100-199 | 2/100 |

## Problemas completados

| # | Nombre | Archivo |
|---|--------|---------|
| 100 | Constante de Kaprekar | `Kaprekar.java` |
| 101 | Cuadrados diabólicos y esotéricos | `CuadradosMagicos.java` |

## Lenguaje

Todos los problemas se resuelven en **Java**.

## Como ejecutar

```bash
# Compilar
javac Problema.java

# Ejecutar
java Problema < entrada.txt
```
